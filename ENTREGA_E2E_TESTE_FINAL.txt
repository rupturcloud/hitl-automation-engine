POST /api/bubble/token
  Localização: api/routes-bubble.mjs:25-99
  Função: Gera token Base64 JWT para Bubble Inbox
  Validação: Supabase JWT + tenant membership
  Resposta: {bubble_url, token, expires_in: 3600}
  Status: ✅ OPERACIONAL

POST /api/bubble/validate
  Localização: api/routes-bubble.mjs:116-417
  Funções:
    1) Validar token Bubble (header X-Token)
    2) Processar webhooks UAZAPI (body com event, instance_id, data)
  Validação: Token expiry, tenant membership
  Resposta: 200 {valid:true} ou 201 {success:true, id, ...}
  Status: ✅ OPERACIONAL

POST /api/messages/send
  Localização: api/routes-messages.mjs:30-114
  Função: Enviar mensagem via UAZAPI
  Validação: JWT extraction, fields validation
  Resposta: {success:true, message_id, status:pending}
  Status: ✅ OPERACIONAL
