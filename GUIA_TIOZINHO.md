# 🎮 Will Bot - Guia do Tiozinho

## ⬇️ PASSO 1: Abrir o Terminal

Procure no seu Mac:
- Clique em **Spotlight** (canto superior direito, lupa)
- Digite: `terminal`
- Clique em **Terminal**

Uma janela preta vai abrir. Pronto!

---

## 📝 PASSO 2: Copiar e Colar o Comando

**Para DIEGO:**

Copie este comando inteiro:
```
python3 /Users/diego/dev/ruptur-cloud/will_bot_diego.py
```

- Cole no terminal (Cmd+V)
- Pressione **ENTER**

**Para LETICIA:**

Copie este comando inteiro:
```
python3 /Users/diego/dev/ruptur-cloud/will_bot_leticia.py
```

- Cole no terminal (Cmd+V)
- Pressione **ENTER**

---

## 🌍 PASSO 3: O Navegador Abre

Um navegador do Chrome vai abrir sozinho com:
- **BetBoom carregando** 🎰
- **Extensão pronta** ⚡

Deixe aberto!

---

## 👤 PASSO 4: Fazer Login

Você vê a tela de login do BetBoom:

1. Clique em **"Entrar com Google"**
2. Digite seu **email** (ex: diegoizac@gmail.com)
3. Digite sua **senha**
4. Clique **"Entrar"**

**⏳ Você tem 120 SEGUNDOS para fazer login!**

Depois o sistema salva automaticamente.

---

## ✅ PASSO 5: Confirmação

No terminal você vai ver:

```
[HH:MM:SS][DIEGO] ✅ 27 cookies salvos

╔══════════════════════════════════════╗
║  ✅ PRONTO PARA RODAR!               ║
║                                      ║
║  Clique [GO] na extensão            ║
║  Robot executará automaticamente!    ║
╚══════════════════════════════════════╝
```

**Pronto!** Sessão salva! 🎉

---

## 🚀 PASSO 6: Rodar o Robot

Agora você vê a **Extensão** no canto superior do navegador.

1. **Clique no ícone da extensão** (caninha no canto)
2. Um painel abre
3. Clique no botão **[GO]** 🟢
4. **PRONTO!** Robot começa automaticamente!

---

## 📋 Resumo Rápido

```
1. Terminal → copiar comando
2. Pressionar ENTER
3. Navegador abre sozinho
4. Fazer login (120s)
5. Clique [GO] na extensão
6. Fim!
```

## ❓ Dúvidas

**P: E se perder?**  
R: Tudo está no navegador. Se fechar, roda o comando de novo.

**P: Posso mudar a senha?**  
R: Sim, roda o comando de novo e faz login com a senha nova.

**P: Funciona em outro computador?**  
R: Sim! O script está pronto pra usar em qualquer Mac.

**P: Precisa de internet?**  
R: Sim, BetBoom precisa de internet.
