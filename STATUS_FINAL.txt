 TROUBLESHOOTING RÁPIDO
═══════════════════════════════════════════════════════════════════════════════

❌ PROBLEMA: ws_capture_test.json não tem dados

CAUSA: Extensão não carregada

SOLUÇÃO:
  1. Chrome → Extensões (chrome://extensions)
  2. Procurar "Will Dados Pro"
  3. Garantir que está ATIVADA (azul)
  4. Clicar em "Reload" se necessário
  5. Abrir DevTools (F12) → Console
  6. Digitar: window.__BETIA
  7. Se undefined, extensão não está no contexto certo


❌ PROBLEMA: Clique não funciona

CAUSA: Evolution usa Canvas, elemento não é clicável

SOLUÇÃO:
  Usar JavaScript force-click no script
  Mudar de: sb.click("//button[BANKER]")
  Para: sb.execute_script("""
    document.querySelectorAll('*').forEach(el => {
      if (el.textContent.includes('BANKER')) el.click();
    });
  """)


❌ PROBLEMA: Balance não atualiza

CAUSA: wsData.balance vem em string ("R$ 1.000,00")

SOLUÇÃO:
  Usar parser de número:
  balance = float(str_balance.replace("R$ ", "").replace(".", "").replace(",", "."))


❌ PROBLEMA: "Sessão expirada"

CAUSA: Cookies do BetBoom expiraram (válidos por ~24h)

SOLUÇÃO:
  python3 save_session.py  # Fazer login novamente (1x)
  Agora cookies estão frescos
