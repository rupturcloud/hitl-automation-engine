## 📌 O Que é Bac Bo?

**Bac Bo** é um jogo de cassino ao vivo criado pela Evolution Gaming. É uma evolução criativa que combina elementos do clássico **Baccarat** com a mecânica de **dados (Sic Bo)**, criando uma experiência única e emocionante.

### Características Principais
- 🎯 Jogo ao vivo com dealer em tempo real
- 🎲 Resultado definido por dados (dados físicos)
- 💰 Altas margens favoráveis ao jogador
- ⚡ Rodadas rápidas (30-45 segundos)
- 📊 RTP de até 98,87%

## 🎮 Como Funciona

### A Estrutura Básica

A mesa de Bac Bo tem **três opções de aposta**:

```
┌─────────────────────────────────────┐
│                                     │
│    JOGADOR    │    EMPATE    │    BANQUEIRO  │
│                                     │
└─────────────────────────────────────┘
```

### O Fluxo de Uma Rodada

```
1. ABERTURA (13 segundos)
   └─ Você escolhe sua aposta (JOGADOR, EMPATE ou BANQUEIRO)

2. FECHAMENTO
   └─ Nenhuma aposta adicional permitida

3. LANÇAMENTO DOS DADOS
   └─ Dealer lança 2 dados para JOGADOR
   └─ Dealer lança 2 dados para BANQUEIRO

4. CÁLCULO DO RESULTADO
   └─ Soma os dados (ex: 4 + 3 = 7 pontos)
   └─ Compara JOGADOR vs BANQUEIRO

5. PAGAMENTO
   └─ Prêmios distribuídos conforme resultado
```

## 🎲 Sistema de Pontos

### Como Contar os Dados

Cada dado mostra um número de **1 a 6**. A **soma dos dois dados** é sua pontuação final.

#### Exemplos de Contagem

| Dados | Soma | Pontos |
|-------|------|--------|
| 1 + 1 | 2 | 2 |
| 3 + 4 | 7 | 7 |
| 5 + 6 | 11 | **1** (descartar dezena) |
| 6 + 6 | 12 | **2** (descartar dezena) |

### Regra: Descartar a Dezena

Se a soma der **10 ou mais**, você **descarta a dezena** e fica apenas com a unidade:
- 10 = 0 pontos
- 11 = 1 ponto
- 12 = 2 pontos
- ...
- 17 = 7 pontos
- 18 = 8 pontos (máximo)

**Exemplo Real:**
```
JOGADOR: 6 + 6 = 12 → descarta 1 → 2 pontos
BANQUEIRO: 5 + 5 = 10 → descarta 1 → 0 pontos

JOGADOR GANHA (2 > 0)
```
