🔄 FLUXO END-TO-END VALIDADO

1. Usuário acessa /inbox
   ↓ useAuth() carrega session (JWT Supabase)
   ↓
2. Inbox.jsx chama POST /api/bubble/token
   ↓ Auth header: Bearer <JWT>
   ↓ Retorna: { bubble_url, token, expires_in }
   ↓
3. Bubble iframe carrega com token query param
   ↓ Sandbox: allow-scripts, allow-forms, camera, mic
   ↓
4. Bubble chama POST /api/bubble/validate (page load)
   ↓ Header: X-Token <base64_token>
   ↓ Retorna: { valid, user_id, tenant_id }
   ↓
5. Sidebar carrega chats (paralelo)
   ↓ Query: uazapi_chats WHERE tenant_id = ?
   ↓ Subscribe realtime para mudanças
   ↓
6. Webhook UAZAPI chega
   ↓ POST /api/bubble/validate com { event, instance_id, data }
   ↓ Valida token, verifica membership
   ↓ Insere em uazapi_messages
   ↓ Pub/Sub notifica subscribers
   ↓ UI sincroniza
